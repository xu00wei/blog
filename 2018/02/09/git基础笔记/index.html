使用git管理文件夹

git init

git 工作流

工作区

  • 工作区到缓存区 git add /path/to/file
  • 工作区到不需要跟踪 git rm --cached /path/to/file
  • 工作区取消修改 git checkout -- /path/to/file

缓存区

  • 缓存区到仓库区 git commit -m "blalalala"
  • 缓存区到工作区 git reset -- /path/to/file (–hard 会消除所有修改)

仓库区

  • 仓库区到工作区 git reset 哈希码（将几个仓库版本的变动合起来，慎用）
  • 仓库区回滚 git reset --hard HEAD^ (HEAD代表当前分支)

分支处理

  • 查看指令：git branch --help（分支的增删改查）
  • 合并分支：git merge 分支名
  • 切换分支：git checkout 分支名
  • 新建并切换分支：git checkout -b 分支名

查看提交

git log --help

和远程交互

远程两种交互途径：
ssh, http

ssh使用说明：

  • 生成公私钥: ssh-keygen
  • 将.pub文件内容上传自己的git 设置中(pub即公钥)

http使用说明:

输入账号密码即可

远程仓库操作

查看指令： git remote --help

  • 添加远程仓库： git remote add 远程名(origin) 远程地址
  • 下拉远程代码： git fetch 远程名 分支名
  • 下拉并自动合并代码：git pull 远程名 分支名 ==》 git fetch 后 git merge
  • 推送本地仓库到远程：git push 远程名 分支名
